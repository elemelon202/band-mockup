# THUNDERSTRIKE - Band Website Mockup

A modern, animated promotional website mockup for the fictional rock band "THUNDERSTRIKE".

## Features

- **Hero Section** - Full-screen landing with animated band name and call-to-action buttons
- **Band Members** - Profile cards for each member with role and bio
- **Gallery** - Grid layout showcasing concert and behind-the-scenes content
- **Merchandise Store** - Product cards with pricing and add-to-cart buttons
- **Contact Form** - Booking inquiries with categorized message types
- **Responsive Design** - Mobile-friendly layout with breakpoints for tablet and phone

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox, Animations)
- JavaScript (ES6+)
- [GSAP](https://greensock.com/gsap/) - Animation library with ScrollTrigger plugin
- Google Fonts (Bebas Neue, Oswald, Noto Sans JP)

## Animations

The site includes various GSAP-powered animations:

- Page load sequence for hero content
- Scroll-triggered reveals for sections
- Parallax effect on hero
- Magnetic button hover effects
- Custom cursor with hover states
- Click spark effects
- Glitch effect on section titles

## Getting Started

1. Clone the repository
2. Open `index.html` in a browser

No build process required - this is a static site.
